# 415. Add Strings

https://leetcode.com/problems/add-strings/

Basic Idea: The idea is very naive by using Python


```python=

class Solution:
    def addStrings(self, num1: str, num2: str) -> str:
        answer = int(num1) + int(num2)
        return str(answer)

```
