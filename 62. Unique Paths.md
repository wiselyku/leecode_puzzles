# 62. Unique Paths

Basic Idea: The number of Unique Paths is the same to the combination number of C(n-1+m-1, n-1)


```python=
from math import comb

class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        if m == 1 or n == 1:
            return 1
        
        else:
            m = m - 1
            n = n - 1
            return comb( m + n, n )
            
```
