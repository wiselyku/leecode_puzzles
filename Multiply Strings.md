# 43. Multiply Strings

https://leetcode.com/problems/multiply-strings/

Basic Idea: It is very naive if you write it in Python.


```python=

class Solution:
    def multiply(self, num1: str, num2: str) -> str:
        answer = int(num1)*int(num2)
        return str(answer)

```
